# fareShare

A collaborative expense sharing application for managing shared costs among friends and groups.

## Features

- Create and manage shared expenses
- Split costs equally or custom amounts
- Track who owes whom
- Generate settlement reports
- User-friendly interface

## Installation

```bash
git clone https://github.com/yourusername/fareShare.git
cd fareShare
npm install
```

## Usage

```bash
npm start
```
